
### Aritmetic
- `5/3 -> 1.66666` : return the whole part
- `Math.floor(5/3) -> 1` -> round the Z+ part
- `5%3 -> 2` -> return the rest part
### Concepts
- Returning null or undefined values
  - `let x:number` -> by default is **undefined**, it was never assigned to it.
  - `let x:number|null =  null` -> A manual process.
